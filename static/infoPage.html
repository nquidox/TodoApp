HTTP codes

2xx

Код Описание Причины
200 OK Если не возникло никаких ошибок.
204 No content

Все ок, но возвращать нечего.

Объекты в базе данных не найдены, но это не ошибка.

4xx

КодОписаниеПричины
400 Bad request

/me Ошибка чтения пользователя

/login Ошибки валидации - некорректный емейл, пустой пароль. Пароли не совпадают.

Во многих запросах:

Ошибка при чтении куки.

Ошибка при чтении тела запроса.

Некорректный uuid листа и/или таска.
401 Unauthorized Нет токена авторизации
403 Forbidden /login При попытке зайти в систему без подтвержденного емаила.
404 Not Found В многих запросах - запись не найдена. В большинстве случаев попытка изменить запись, которая была изменена/удалена.
409 Conflict /login Попытка зарегистрировать пользователя с уже существующим в БД емаилом.
410 Gone Попытка верифицировать почту с уже истекшим сроком действия ссылки.
422 Unprocessable entity Некорректный формат полученного json-а.
